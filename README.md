# EmergencyLights
Sistema de control de Luces de emergencia escolares
Se encargará del control de carga de una bateria de 14v
Debe mantener su carga dentro del rango de 12v (carga minima) y 13.5 (Carga maxima)

En caso de cortarse el suministro electrico, se deberan activar las luces de emergencia si se tratará de un dia de semana (lunes a viernes) \n 
y el corte sea dentro de un horario de clase de poca luminosidad (7AM-12AM horario mañana y 17-22.30 horario vespertino)
